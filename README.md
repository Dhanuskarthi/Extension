# 🎯 Overview

**siuuuuu** is a powerful Chrome extension that leverages AI to provide **instant quiz answers** and **voice-based Q&A**. Built with modern web technologies and designed for speed, privacy, and ease of use.

The extension supports multiple AI providers including **Gemini, OpenAI, Claude, and Grok**, allowing users to choose their preferred AI service.

---

## 🚀 Why siuuuuu?

- 🚀 **Lightning Fast** - Instant answer detection and selection
- 🎨 **Premium UI** - Modern, glassmorphic overlay with dark/light themes
- 🔒 **Privacy First** - No user data collection
- 🌐 **Multi-Platform** - Supports Moodle, Google Forms, Canvas, Study4, and more
- 🤖 **Multi-AI Support** - Gemini, OpenAI, Claude, and Grok
- 🎤 **Voice Assistant** - Ask questions using your microphone
- 💾 **Answer Caching** - Reduces API requests and improves speed

---

# 🔑 API Key Required

> ⚠️ **A valid API key is required to use AI-powered features.**

The extension does not include built-in AI access. Users must provide their own API key from one of the supported providers.

## Supported AI Providers

| Provider | Supported |
|----------|------------|
| Gemini | ✅ |
| OpenAI | ✅ |
| Claude | ✅ |
| Grok | ✅ |

## Setup Instructions

1. Obtain an API key from your preferred AI provider.
2. Open the extension settings.
3. Select the AI provider.
4. Paste your API key.
5. Save settings.
6. Start using Quiz Automation and Voice Q&A features.

## Privacy

- API keys are stored locally in your browser.
- API keys are never shared with third-party services.
- Requests are sent only to your selected AI provider.

## Important

Without a valid API key:

❌ Quiz Automation will not work

❌ Voice Q&A will not work

❌ AI-generated explanations will not work

# ⚙️ Settings Reference

| Setting | Default | Description |
|----------|----------|-------------|
| Auto Detect | ON | Detect quiz questions automatically |
| Auto Submit | OFF | Submit quizzes automatically |
| Show Explanation | OFF | Display answer explanations |
| Cache TTL | 24 Hours | Cache answer duration |
| Click Delay | 50-200ms | Human-like click timing |
| Theme | System | Dark, Light, or System |
| AI Provider | Gemini | Select AI backend |
| API Key | Required | Authentication for AI provider |
